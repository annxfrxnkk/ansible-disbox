### Prerequisites

Install distrobox on your specific distrobution 
* Arch Linux
  
  ```sh
  sudo pacman -S distrobox podman
  ```
* Debian/Ubuntu
  
  ```sh
  sudo apt install distrobox -y
  ```
* Fedora
  
  ```sh
  sudo dnf install distrobox
  ```
